# code_the_dream_pre_work
This App is a web application using vanilla javascript , html and css that provide information about the Football/Soccer information of the two of the most popular leagues in the world 1) English Premier League  2) La Liga . It can help fetch the infomation about the players , view stats and injuries status for a fixture . We can toggle between both the leagues 

#browser fixtures from EPL and la liga leagues 
#view match predictions from the fixtures
#search players from the leagues 
#get the injured players list 
#toggle between the leagues 

#Installation 
clone the application repository  from github  link below : 

https://github.com/Neosteel/code_the_dream_pre_work.git


Navigate to project directory 

cd code_the_dream_pre_work


Open index.html in your preferred browser or run

## Usage
On the Fixtures page click English Premier League or Spanish La Liga to load matches
Click Predictions on any fixture card to view predictions and injuries for that match
Navigate to Players page using the top navigation bar
Select a league, type a player name and click Search
View player card showing goals, assists and card statistics

## API
This app uses API-Football v3
